# Landing

Проект подготовлен для работы с Figma MCP в Codex.

## Что уже подключено

- В глобальный конфиг Codex добавлен официальный Figma MCP:
  - `figma -> https://mcp.figma.com/mcp`
- Плагин `figma@openai-curated` уже включен в Codex.
- Старый локальный сервер `figma_desktop` сохранен и не затронут.

## Что нужно сделать один раз

1. Завершить OAuth-авторизацию Figma в открывшемся браузере.
2. После авторизации перезапустить текущую сессию Codex, если Figma не появится сразу.

## Как будем работать дальше

1. Открываем нужный файл/страницу в Figma.
2. Даешь ссылку на Figma-файл или конкретный frame.
3. Я считываю структуру, токены, размеры, типографику и компоненты через MCP.
4. После этого переношу дизайн в код сайта и проверяю соответствие макету.

## Рекомендуемый workflow для лендинга

- Сначала поднимаем каркас проекта.
- Затем переносим секции по очереди:
  - hero
  - social proof
  - features
  - CTA / footer
- После каждой секции сверяем отступы, типографику, адаптив и состояния.
